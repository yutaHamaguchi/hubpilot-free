# HubPilot Free - SEO記事作成エージェント

## 🎉 Phase 1 完成！

**HubPilot Free**は、HubSpot風のトピッククラスターモデルに基づくSEO記事作成エージェントです。Phase 1（インターフェース）が100%完成し、プロフェッショナルグレードのユーザーエクスペリエンスを提供します。

## ✨ 主要機能

### 📝 6ステップウィザード
1. **テーマ設定** - メインテーマの入力と例示
2. **構成案確認** - ピラーページとクラスターページの生成・編集
3. **見出し構成** - 各記事の見出し設定と管理
4. **記事執筆** - リアルタイム進捗表示と記事生成シミュレーション
5. **品質チェック** - 自動品質評価と修正提案
6. **最終承認** - ピラーページプレビューと公開オプション

### 🎨 デザイン・UX
- **HubSpot風デザイン** - プロフェッショナルなオレンジ×ネイビーカラーパレット
- **完全レスポンシブ** - デスクトップ、タブレット、モバイル対応
- **タッチ操作対応** - スワイプジェスチャー、タッチフィードバック
- **アクセシビリティ** - キーボードナビゲーション、スクリーンリーダー対応

### 💾 データ管理
- **自動保存** - 30秒間隔の自動保存とリアルタイム同期
- **バックアップ機能** - 最新5件の自動バックアップ
- **エクスポート/インポート** - JSON形式でのデータ交換
- **バージョン管理** - データ構造の変更に対応した自動マイグレーション

### 🔧 開発者機能
- **包括的テストスイート** - 自動テストとパフォーマンス監視
- **デバッグコンソール** - 開発者向けコマンドライン
- **品質評価システム** - アクセシビリティとパフォーマンスの自動評価
- **エラーハンドリング** - 堅牢なエラー処理とユーザーフィードバック

## 🚀 使用方法

### 基本的な使い方
1. `index.html` をブラウザで開く
2. Step 1でメインテーマを入力
3. ウィザードに従って各ステップを進める
4. Step 6で完成した記事群をダウンロードまたはCMSに投稿

### 開発者コマンド
ブラウザのコンソールで以下のコマンドが利用可能：

```javascript
// デバッグ情報の表示
hubpilot.debug()

// アプリケーション統計
hubpilot.stats()

// ヘルスチェック
hubpilot.health()

// 包括的テストの実行
hubpilot.test()

// 品質評価
hubpilot.quality()

// アクセシビリティチェック
hubpilot.accessibility()

// 指定ステップへの移動
hubpilot.goToStep(3)

// データのエクスポート
hubpilot.export()

// データのリセット
hubpilot.reset()
```
