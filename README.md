# HubPilot Free - SEO記事作成エージェント

HubSpotのトピッククラスターモデルに基づいたSEO記事作成エージェントアプリケーションのプロトタイプです。

## 概要

このアプリケーションは、コンテンツマーケティング戦略を自動化し、効果的なSEO記事群を生成するためのツールです。ピラーページと複数のクラスターページを含む包括的なコンテンツ戦略を6つのステップで作成できます。

## 機能

### フェーズ1（現在実装済み）
- ✅ ウィザード形式のユーザーインターフェース
- ✅ 6ステップのナビゲーション
- ✅ HubSpot風のクリーンなデザイン
- ✅ レスポンシブデザイン対応
- ✅ ローカルストレージによるデータ永続化
- ✅ モックデータを使用したUI表示

### フェーズ2（実装予定）
- 🔄 実際のAIコンテンツ生成機能
- 🔄 テーマからの自動タイトル生成
- 🔄 記事本文の自動生成
- 🔄 品質チェックアルゴリズム
- 🔄 CMSへの実際の投稿機能

## 6つのステップ

1. **テーマ設定** - メインテーマの入力
2. **構成案確認** - ピラーページとクラスターページの構成
3. **見出し構成** - 各記事の見出し設定
4. **記事執筆** - コンテンツ生成の進捗表示
5. **品質チェック** - 記事の品質確認
6. **最終承認** - ピラーページの統合と完成

## 技術スタック

- **HTML5** - セマンティックなマークアップ
- **CSS3** - HubSpot風デザインシステム
- **Vanilla JavaScript** - フレームワークレスの実装
- **LocalStorage** - データ永続化

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/NADJA-japan/hubpilot-free.git
cd hubpilot-free
```

2. ブラウザでindex.htmlを開く
```bash
# 任意のHTTPサーバーで起動（推奨）
python -m http.server 8000
# または
npx serve .
```

3. ブラウザで `http://localhost:8000` にアクセス

## 使用方法

1. **テーマ入力**: 「Instagramマーケティング」などのメインテーマを入力
2. **構成確認**: 生成されたピラーページとクラスターページのタイトルを確認・編集
3. **見出し設定**: 各記事の見出し構成を確認・調整
4. **記事生成**: 10記事の生成進捗を監視
5. **品質チェック**: 生成された記事の品質を確認
6. **最終確認**: 完成したピラーページを確認し、ダウンロードまたは投稿
